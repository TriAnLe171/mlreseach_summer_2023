# mlresearch
Machine Learning Research Repo

This repo comprises:
1) Simple CNNs models on Arabic and Western Handwritten Dataset
2) Advanced CNNs model like YOLOv1 and YOLOv8
3) Simple RNNs models on Sunspot Dataset
4) Advanced BiLSTM models on IMDB Review Dataset
5) GANs on MNIST handwritten Dataset, CelebA Dataset
6) Minimax on Tic Tak Toe game
